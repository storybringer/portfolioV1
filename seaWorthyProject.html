SeaWorthy Boats

Skills:

  • HTML/CSS/Web Development
  • Teamwork/Agile Practices
  • Git Bash
  • Azure DevOps

Description:

This was made during my second semester of my Collaborative Application Development course, working with my classmates from the school. We created this cool and responsive website for a fictional business "SeaWorthy Boats." For the website, we focused on web development and learning how to follow through with planned build practices. We also learned how to use Git bash to push our changes to the database in Azure DevOps, where we would then release our finished website.
