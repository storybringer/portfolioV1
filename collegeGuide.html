College Student's Basic Guide to Web Development

Skills:

  • HTML/CSS/Web Development
  • Git Bash
  • Azure DevOps
  • Teamwork Agile Practices

Description:

This was made during my second semester of my Collaborative Application Development course, working with my classmates from the school. This was one of our beginner projects where we had to make a website for our choice. Being in college to become web developers, our team decided to make a website that gave a brief overview on the three main tools used for front end Web Development. For the website, we mainly focused on learning how to assign story points/tasks and communicated what we wanted or needed to be done as a team. This was mainly a project to strengthen our agile practices/teamwork as well as getting familar to use Git Bash and Azure Devops.
